🎯 Objective

This project demonstrates how household appliances can be controlled using hand gestures, a Raspberry Pi, a USB camera and a 4-channel relay module.

The camera detects the user's fingers. Each finger is assigned to a particular appliance such as an LED, fan, buzzer or water pump.

📋 Procedure

  1. Download all project files from this webpage.
  2. Connect the USB camera to the Raspberry Pi.
  3. Connect the 4-channel relay module according to the connection table.
  4. Connect the LED, fan, buzzer and water pump to the relay output channels.
  5. Open the Terminal on the Raspberry Pi.
  6. Create a Python virtual environment.
  7. Activate the virtual environment.
  8. Install all required Python libraries.
  9. Save the Python program with the name Home_Automation.py.
  10. Run the Python program.
  11. The camera starts and detects the user's hand.
  12. Show a mapped finger in front of the camera.
  13. The corresponding appliance turns ON or OFF.

⚠ Troubleshooting

Problem Solution
Camera not detected Reconnect the USB camera and restart the Raspberry Pi.
Relay not working Check the VCC, GND and GPIO connections.
Appliance not turning ON Check the relay output connection and external power supply.
Python module not found Activate the virtual environment and install the requirements.
Wrong appliance activates Verify the GPIO and finger mapping table.
Hand not detected Improve room lighting and keep the hand clearly visible to the camera.
